# Welcome to AccessRC Documentation

AccessRC is designed to streamline the process of managing and delivering humanitarian assistance globally. This documentation provides essential information for integrating with AccessRC's APIs, including **authentication**, **webhooks**, and managing **delivery channels** and **delivery requests**.

## Key Features:
- **Authentication**: Secure your API calls with our authentication mechanisms.
- **Webhooks**: Subscribe to event types and verify webhook requests.
- **Delivery Channels**: Create and manage delivery channels for your program.
- **Delivery Requests**: Sync and manage delivery requests for beneficiaries.

For more details, check out the [API Reference](../../api-reference).
